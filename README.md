# <p align="center">B.A.D.A.S.S</p>

> Bgp At Doors of Autonomous Systems is Simple
>
> Le but de ce projet est d’approfondir les connaissances acquises via NetPractice.
> Vous devrez simuler un réseau et le configurer en utilisant GNS3 avec des images Docker.
>
> BGP EVPN est basé sur BGP (RFC 4271) et ses extensions, MP-BGP (RFC 4760).
> BGP est le protocole de routage qui fait fonctionner Internet.
> Grâce aux extensions MP-BGP, il peut être utilisé pour transporter des informations de connectivité (NLRI) pour divers protocoles (IPv4, IPv6, VPN L3 et, dans ce cas, EVPN).
> EVPN est une famille spéciale utilisée pour publier des informations sur les adresses MAC et les équipements terminaux qui y accèdent.
